It features advanced search, filters, and detailed college profiles with info like fees, cutoff, placements, and reviews. Built with HTML, CSS, and JavaScript, it offers an intuitive, Netflix-style browsing experience.

ShopEase

ShopEase is a feature-rich e-commerce web app inspired by Flipkart, offering multiple product categories like Electronics, Fashion, and Books with dynamic filters, animated transitions, and responsive design. It includes Firebase-based login/signup, a localStorage-powered cart, and a simulated checkout system for a complete shopping experience.

Tech: HTML, CSS, JavaScript, Firebase

GitHub | Live Demo

QuizFunFact

QuizFunFact is a responsive JavaScript web project that combines a knowledge quiz with a fun facts section featuring one-click jokes for an engaging user experience. It includes a real-time score tracker, countdown timer, and dynamic question generation, all styled with modern CSS effects to enhance interactivity and learning.

Tech: HTML, CSS, JavaScript
